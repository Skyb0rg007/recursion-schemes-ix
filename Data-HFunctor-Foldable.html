Destructive Morphisms

cata :: forall i h a. HRecursive i h => HFunction i (h a) a -> HFunction i Identity a Source #

Catamorphism: ∀a. (f a → a) → μf → a Deconstructs an f-structure level-by-level and applies the algebra

para :: forall i h a. HRecursive i h => HFunction i (h (Product Identity a)) a -> HFunction i Identity a Source #

Paramorphism: ∀a. (f (μf * a) → a) → μf → a Like cata, but allows access to the full subtree during teardown

zygo :: HRecursive i h => HFunction i (h b) b -> HFunction i (h (Product b a)) a -> HFunction i Identity a Source #

Zygomorphism: ∀a b. (f b → b) → (f (b * a) → a) → μf → a Allows depending on a helper algebra for deconstructing an f-structure

histo :: HRecursive i h => HFunction i (h (HCofree h a)) a -> HFunction i Identity a Source #

Histomorphism: ∀a. (f (Cofree f a) → a) → μf → a
